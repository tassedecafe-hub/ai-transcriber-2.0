# AI Transcriber 2.0

Десктопное приложение для транскрипции речи в реальном времени.  
Позволяет записывать голос с микрофона и преобразовывать его в текст, а также копировать, сохранять и скачивать результат.

---

## 🚀 Возможности

- **Запись с микрофона** — распознавание речи в реальном времени (Web Speech API).
- **Выбор источника** — микрофон, системный звук, аудиофайл, видеофайл (интерфейс готов, бэкенд в разработке).
- **Управление транскрипцией** — копирование, очистка, скачивание в `.txt`.
- **Статистика** — количество слов, символов, время записи.
- **Кроссплатформенность** — работает на Windows (сборка под macOS и Linux планируется).

## 📦 Установка и запуск

### Требования
- Node.js (v18 или выше)
- npm (или yarn)

### Шаги

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/ВАШ_ЛОГИН/ai-transcriber-2.0.git
   cd ai-transcriber-2.0
Установите зависимости

bash
npm install
Запустите сервер разработки Vite

bash
npm run dev
Сервер будет доступен по адресу http://localhost:5173.

Запустите Electron (в отдельном терминале)

bash
npx electron start.js
Или, если у вас настроен скрипт:

bash
npm run electron:dev
🖥️ Как пользоваться
Выберите источник: Микрофон (другие источники пока в разработке).

Нажмите «Начать» — приложение запросит доступ к микрофону.

Говорите в микрофон — текст будет появляться в поле транскрипции.

Нажмите «Остановить», чтобы завершить запись.

Используйте кнопки: Копировать, Очистить, Скачать TXT.
